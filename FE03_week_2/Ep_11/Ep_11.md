### Whats the difference between state and prop

- state is local variable
- props is the arguement
  - its used to pass data from one component to another component

### Props drilling

- suppose you have state variable in a top most level component
  - and you want to pass those state variables to bottom most level components, how do you do it?
    - one of the ways would be to, passing down the props from one component to another to another in hierarchy format, this is prop drilling

- one of the disadvantages of prop dilling

### Lifting the state up

- suppose you have 3 sections which have text, with buttons show and hide, like accordion
- each section has a state, which allows the show and hide feature
- now you want to implement feature that, whenever user clicks on "show" on one of the section, you want other sections to close down, for that you need to put states in each of the section to its parent component, that is lifting the state up
