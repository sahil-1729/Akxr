### useRef
- it returns a object, which provides the reference to the element we refered
- it is initialized like this 
    - const email = useRef(null)
        - setting the intial default value as null
    - using it at the tag
    - <div ref={email}></div>
    - to access it we use email.current 
        - .current refers to the current referred object or element
- it is used to refer a field or a tag here, like input field
